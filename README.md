# Pet-Adopt

O objetivo do projeto é criar uma plataforma que permita um usuário adotar, apadrinhar, oferecer lar temporário ou reivindicar o seu pet. Os pets no catálogo podem ser de várias ONGs, que  estarão previamente cadastradas no sistema.
